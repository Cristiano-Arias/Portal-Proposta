# 🏢 Sistema de Gestão de Propostas

Sistema web completo para gerenciamento de processos licitatórios e propostas comerciais, com envio automático de e-mails e geração de relatórios.

## 📋 Funcionalidades

- ✅ **Gestão de Processos Licitatórios**
  - Cadastro e acompanhamento de processos
  - Controle de prazos e modalidades
  - Geração de links para portal de propostas

- 📝 **Portal de Propostas**
  - Interface completa para fornecedores
  - Preenchimento de propostas técnicas e comerciais
  - Cálculo automático de valores e BDI
  - Envio automático por e-mail

- 📊 **Relatórios e Análises**
  - Comparativo de propostas com IA
  - Exportação para Excel e PDF
  - Dashboard com estatísticas

- 👥 **Sistema Multi-usuário**
  - Admin, Compradores e Fornecedores
  - Controle de acesso por níveis
  - Autenticação e logs de atividade

## 🚀 Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Python 3.8+ com Flask
- **Email**: SMTP com Flask-Mail
- **Armazenamento**: LocalStorage (desenvolvimento)
- **Deploy**: Render.com
